# SauceDemo Automation Testing Project

## Project Overview

This project automates functional testing for the SauceDemo website using Selenium WebDriver and TestNG

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager
- Git

## Test Coverage

- Login (valid, invalid, empty, locked user)
- Cart (add/remove product)
- Checkout (success and validation cases)
- Sort products (name and price)
- Logout

## Testing Techniques Applied

- Equivalence Partitioning
- Boundary Value Analysis
- Negative Testing

## How to Run

1. Clone the repository
2. Open project in IntelliJ
3. Run:
   mvn test
